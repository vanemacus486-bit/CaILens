# CaILens 隐私政策 (Privacy Policy)

**最后更新：2026-06**

本隐私政策说明 CaILens 桌面及移动端应用（"本应用"）如何处理用户信息。我们高度重视您的隐私，并致力于保持透明。

---

## 1. 信息收集

**我们不收集任何个人信息。** 本应用的设计理念是"本地优先"：

- **无账号系统** — 您无需注册账号即可使用本应用的全部功能。
- **无服务器** — 本应用没有后端服务器。所有数据存储于您本地设备上的 IndexedDB 数据库和/或本地文件系统中。
- **无遥测** — 我们不进行任何形式的用户行为追踪、使用分析或崩溃报告收集。

您在本应用中创建的所有内容（事件记录、分类、待办事项、个人资料等）**完全归属于您**，且仅存在于您的设备上。

## 2. 网络请求

本应用在以下情况下会发起网络请求：

### 2.1 检查更新（桌面端 Tauri）

桌面端启动时或在设置页手动触发时，会向 GitHub Releases API 发送一次请求：

```
GET https://api.github.com/repos/vanemacus486-bit/CaILens/releases/latest
```

此请求**仅获取最新版本号**，用于判断是否有新版本可更新。请求中**不包含任何用户数据、设备标识符或个人信息**。此为 GitHub 公共 API，其自身隐私政策请参阅 [GitHub 隐私声明](https://docs.github.com/zh/site-policy/privacy-policies/github-general-privacy-statement)。

### 2.2 字体加载

本应用使用的所有字体均通过 `@fontsource` 以 npm 包形式本地打包，**不连接任何 CDN**。字体文件在构建时内嵌于应用中，运行时不会向外部服务器发起字体请求。

### 2.3 第三方链接

"设置 → 支持 / 赞助"页面中包含跳转至第三方平台（爱发电、Gumroad 等）的链接。点击这些链接后，您将离开本应用并受相应第三方平台的隐私政策约束。我们不对第三方的隐私实践负责。

## 3. 数据存储与安全

### 3.1 存储位置

- **桌面端（Tauri）**：数据存储于本地 IndexedDB 和/或应用数据目录中的文件。具体路径取决于操作系统。
- **移动端（Capacitor）**：数据存储于移动设备的本地 IndexedDB。

### 3.2 数据导出

本应用支持用户主动将数据导出为文件（JSON/CSV 格式及加密导出）。导出的文件保存于用户指定的位置。导出行为完全由用户发起和控制。

### 3.3 安全

因所有数据仅存储在本地，数据安全取决于您的设备安全。我们建议：

- 保持操作系统和安全软件更新
- 不与他人共享您的设备账户
- 定期备份重要数据（使用应用内的导出功能）

## 4. 儿童隐私

本应用不面向 13 岁以下儿童，且因我们不收集任何信息，我们无法知悉用户的年龄。

## 5. 广告与分析

本应用**不含任何广告**，**不集成任何第三方分析 SDK**。

## 6. 您的权利

因我们不持有您的任何数据，所有数据管理权均在您手中：

- **访问与导出**：您可在应用中随时导出您的数据。
- **删除**：您可随时删除应用中的任意记录，或卸载应用以彻底清除所有本地数据。

## 7. 政策更新

我们可能不时更新本隐私政策。更新后的政策将随新版本应用一同分发。重大变更时，我们会在应用启动时提供醒目通知。
