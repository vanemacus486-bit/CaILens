CaILens

记录你每一小时的去向。

一本你一小时一小时写出来的书。

立即开始 npx cailens — 需要 Node 20+
Philosophy

不是日历，是镜子

市面上几乎所有时间工具都在解决「把未来安排进格子里」——提醒你开会、收邀请、设闹钟。 但理解自己的时间去向，是完全不同的命题。

柳比歇夫从 26 岁开始，每天记录每一小时，持续 56 年直至去世。
这不是效率表演——而是一种真正看清楚生命如何被度过的工具。
—— 亚历山大·亚历山德罗维奇·柳比歇夫，1890–1972

记录，不规划

这里没有日程，没有提醒，没有闹钟。你记录的是已经发生的事——不是对明天的承诺。

本地优先

数据存在 IndexedDB 里。没有账号、没有服务器、没有数据上报。你的时间日记只属于你。

Taxonomy

六个分类，一本账簿

每一条记录都属于一个分类。分类名称可以改，预算可以调——但框架本身是固定的。你不是在自定义标签， 你是在用同一套语言审视自己的生活。

主要矛盾 次要矛盾 庶务时间 个人提升 休息娱乐 睡眠时长

分类体系借用了「矛盾」这个哲学概念——你对自己有判断、有挣扎。应用不替你定义什么「好」什么「坏」，只提供一面镜子。

Highlights

安静的设计，克制的交互

不催促、不评价、不打扰。暖中性色调，衬线标题，克制的焦橙主色。 产品里唯一持续动起来的元素是当前时间红线——因为它代表「现在」。

三视图导航

周 / 日 / 月三视图一键切换。日期头点击跳转日视图，URL 参数驱动——浏览器前进后退键恢复位置。

日视图日记

Serif 阅读版面，Markdown 渲染，分类切换分隔线。不是「dot + time + title」的流水账，而是一页可以读的日记。

拖拽编辑

原生 Pointer Events 手写，分钟轴连续坐标系，ghost 浮层 60fps 渲染。跨天拖拽只是加减法。

标准周视图

把历史上所有周叠加，回答一个问题：周一上午 9 点，我通常在做什么？

统计看板

趋势折线图、睡眠节律散点图、年度热力图。百分位动态阈值，双视图切换，数据成熟度三级生效。

AI 时间助手

多提供商流式对话，@ 提及结构化注入，分析可沉淀到日视图日记。AI 是你的时间数据解读者。

还有：全局快捷键、快速记录、搜索命令面板、ICS 导入、加密备份、深浅色主题……
完整功能列表见 GitHub README →

Install

一行命令，立即开始

# 推荐：npx 总是运行最新版
npx cailens

# 或全局安装
npm install -g cailens
cailens

需要 Node 20 以上。启动后自动打开浏览器，数据保存在本地 IndexedDB。

Stack

经得起时间考验的代码

单向依赖分层，TypeScript 严格模式，489 个测试。技术决策倾向于「少而稳」而不是「多而新」。

前端

状态 & 存储

图表 & 时间

测试
