# CaILens

[English Version](./README.en.md)

> *记录时间。像一本你一小时一小时写出来的书。*

CaILens 是一个本地优先的时间记录工具，灵感来自《奇特的一生》中柳比歇夫坚持了 56 年的"时间统计法"。它记录、分类、可视化你的时间去向——没有账号、没有服务器、没有数据上报。

## 设计哲学

市面上几乎所有日历工具都在解决"把未来安排进格子里"——提醒你开会、收邀请、设闹钟。但理解自己的时间去向是完全不同的命题。

柳比歇夫从 26 岁开始记录每天每一小时，持续 56 年直至去世。这不是"效率表演"——而是一种真正看清楚生命如何被度过的工具。

CaILens 就是这个工具在浏览器里的一个尝试。

- **记录，不规划。** 这里没有日程。你记录的是已经发生的事，不是对明天的承诺。
- **本地优先。** 数据存在 IndexedDB 里。没有账号、没有服务器、没有数据上报。你的时间日记只属于你。
- **安静的设计。** 暖中性色调，衬线标题，克制的主色。应用不催促、不评价、不打扰。
- **代码质量优先于功能数量。** TypeScript 严格模式，489 个测试，单向依赖分层。代码库应该经得起时间考验。

## 功能

### 周视图日历

- **24 小时一屏可见**（周一至周日，带时间刻度栏）。
- **点击空白创建**——事件就地展开为内联编辑卡片（不弹窗、不抢焦点），title / description / location 直接可编辑。再点或点空白处收起，编辑摩擦降到最低。
- **创建后焦点自然流入描述**——标题填完按 Enter 保存，焦点自动落入 description 框；按 Esc 才彻底关闭。写描述是自然的下一步，不是额外的负担。
- **写了描述的事件有视觉重量**——含 description 的事件块左侧色条加厚一档（5px vs 3px）。不用 emoji/角标，只是视觉重量的细微差别——字写得越多越被看见。
- **悬浮快读预览**——hover 事件 ~400ms 后浮现 floating card，显示完整标题 + 描述前两行，移开即消失。不影响点击，只给"扫一眼"的低成本路径。
- **拖拽移动**——分钟轴连续坐标系统，跨天无缝。基于原生 Pointer Events 手写，ghost 浮层 60fps 渲染。
- **拖拽边缘改时长**——上下手柄调整起止时间，可拖过午夜变为跨天事件。
- **跨天事件**——睡眠等跨越午夜的记录自动跨天显示，带箭头指示器和连续圆角。
- **实时预览 60fps**——编辑或拖动时，草稿事件即时渲染。
- **右键菜单**——删除、换颜色（6 种颜色，每种对应一个分类）。
- **重叠事件并排**——自动水平排列。
- **当前时间红线**——1px 实线 + 左侧 3px 圆点 + 2 秒周期呼吸动画（opacity 90%→100%）。产品里唯一允许持续动起来的元素，因为它代表"现在"。
- **浅色 / 深色模式**——手动切换，首次自动检测系统偏好。设置页提供 segment-control 切换。暗色模式下 6 色事件背景/文字经过 WCAG AA 对比度审计，sand 和 stone 在深底色下视觉可区分。
- **三视图导航**——工具栏 W / D / M 三颗 pill 一键切换周/日/月视图。视图状态由 URL 参数驱动（`?view=day` / `?view=month`），浏览器前进/后退按键恢复视图和位置。日期头点击日期自动跳转日视图。

### 标准周视图

把历史上所有周叠加为"典型的一周长什么样"，回答一个问题：*周一上午 9 点，我通常在做什么？*

- **7×24 聚合网格**——复用现有周历网格，168 个 (weekday, hour) 桶按**分钟权重**切片。22:30–23:15 的事件在 22 桶 +30 分钟、23 桶 +15 分钟。
- **时段习惯可视化**——每桶显示最高频活动名称。背景色深 = 习惯强（占比高），色浅 = 混杂。颜色继承分类配色。
- **＜30% 无规律归灰**——低占比格子退至透明背景 + 极浅灰文字，让有规律的块在视觉上跳出。
- **悬停分布列表**——hover 任意块显示完整活动分布（活动名 + 次数 + 占比）。
- **数据范围选择**——全部历史 / 最近 12 周 / 最近 4 周，默认全部。
- **隐藏睡眠开关**——开启后排除「睡眠时长」分类（类别 `stone`），对应格子留空。
- **≤8 周显示绝对次数**——历史数据少时用 `"3/4 周"` 替代百分比，避免 `"75%"` 的虚假精度。

### 快速记录

- **`n` 键全局快捷键**——任意页面打开快速记录对话框；对话框打开时快捷键自动禁用，防止误触。

### 每日生活上下文（v3.22）

CaILens 现在不只是时间记录器——它也开始捕捉"为什么"。

- **轻量每日记录**——在日视图点击「记录」按钮，用 5 个滑条 + 2 个数字输入 + 1 个备注字段记录当天影响作息的生活变量：最后一餐时间与类型、社交强度、户外时长、运动强度、情绪基调、屏幕使用。
- **全字段可选**——只需填你在意的变量，20 秒完成。
- **状态提示**——已记录当天的按钮显示"已记"，未记录则显示"记录"。
- **克制模式**——设置中可开启"只记录不分析"模式，隐藏所有 AI 和洞察入口。
- **极简输入流**——仅保留事件名、分类、备注三个字段。打开即 autofocus 标题框，**Tab** 在事件名和备注之间快速切换，**Enter** 直接保存关闭。
- **Alt+数字选分类**——`Alt+1..6` 随时切换 6 种分类，颜色按钮上也标注了数字快捷键。
- **无需手动设时间**——事件自动使用默认时间段（继承上一事件或回退到 `now - 1h` → `now`），去掉了时间选择器和地点输入，减少决策负担。
- **保存后撤销**——右下角 3 秒 snackbar，点击撤销即删除事件。3 秒内连保存两条，旧 toast 被替换但不影响已保存事件。
- **工具栏入口**——周视图顶部 `+` 按钮，与搜索按钮同排，tooltip 提示快捷键。

### 搜索

- **Ctrl+K / Cmd+K 全局快捷键**——任意页面唤出搜索面板。
- **关键词搜索**——匹配事件标题、描述和地点，大小写不敏感。
- **描述命中显示片段**——命令面板命中 description 时，结果显示命中前后 ±20 字上下文并高亮关键词。让用户感知"我写的字真的能被找回来"。
- **即时导航**——点击搜索结果跳转到对应周并自动打开事件详情卡片。
- **工具栏入口**——周视图工具栏搜索图标按钮。

### 侧边栏

- **图标栏 hover 展开**——200ms 延迟，悬停显示中英文标签。可固定展开状态，状态持久化到 localStorage。
- **周导航**——上一周、下一周、跳回本周。
- **ICS 导入**和**统计看板**入口。

### 日视图日记（DayEventStream）

日视图不再是一条独立路由，而是通过 `?view=day&date=YYYY-MM-DD` URL 参数嵌入到主视图中，与周/月视图共享同一 URL 空间。工具栏 W / D / M 三颗 pill 一键切换。

- **竖排阅读区块**——每条事件不再只是"圆点 + 时间 + 标题"，而是阅读版面：Serif 小标题（600 weight）、Serif 正文段落（描述）、更小的 sans-serif 脚注（地点）。空 description 不显示占位文案——留白即邀请。
- **描述 Markdown 渲染**——输入仍是纯文本，渲染层支持 **加粗** / *斜体* / 行内代码 / 无序列表 / 行内链接。写柳比歇夫式的"p.120-145""信件给 X"更舒服。不引入富文本编辑器。
- **分类切换分隔线**——活动类型变化时出现微妙的分割线。
- **前后日导航**——一天一天翻阅你的日记。周视图日期头点击日期直接跳转日视图。

### 月视图（MonthView）

- **月度聚合网格**——显示当月的每日事件汇总，每格包含当日事件列表概览。
- **双击跳转日视图**——双击任意日期格，自动切换到日视图并定位到该天。
- **前后月导航**——工具栏 `‹` / `›` 箭头切换月份，导航逻辑与周/日视图统一。
- **三视图统一切换**——工具栏 W / D / M 三颗 pill 一键在周、日、月视图间切换。视图状态由 URL 参数驱动（`?view=month&date=2026-05-01`），浏览器前进/后退按键恢复视图和位置。

### 设置页

设置页采用标签式子路由布局，左侧导航栏切换：

- **分类**（默认）——分类名称行内编辑、每周预算滑块（BudgetBar）、关键词文件夹管理（新增/删除/拖拽跨文件夹移动）、关键词变更后自动重归类全库事件。
- **外观**——语言切换（中文 / English）、主题切换（浅色 / 深色，首次自动检测系统偏好）、主题色四选一（rust / ocean / forest / plum）、**字体选择**（默认 Inter / 霞鹜文楷）。
- **快捷键**——查看和自定义所有键盘快捷键，冲突检测警告。
- **数据**——CSV / JSON / **加密 .cailens** 一键导出。**批量粘贴**日视图快速录入。
- **存储**——IndexedDB 存储信息概览。
- **关于**——版本与许可信息。

### 统计看板

点击侧边栏图表图标进入。三视图切换：**趋势 / 睡眠 / 热力图**。

**分类趋势图**
- 支持 **日 / 周 / 月** 三级粒度切换（标题区 pills），标题带 `‹` `›` 箭头前后导航。
- 多线折线图（Recharts），分类 pill 多选，选择项持久化到 localStorage。
- **核心投入组**——accent + sage 一键聚合为堆叠面积 + 粗线总计，独立 toggle。
- 水平虚线标记预算基准线，标签含具体数值（如"预算 30.0h"）。
- Y 轴自适应 + 统计摘要卡（总投入 / 日均 / 高峰分类 / 环比变化）。
- 自动洞察条：分析分类涨跌趋势。

**睡眠节律图**
- 就寝与起床散点图，Y 轴 00–24 日历格式（0 在顶部一天开始，24 在底部一天结束）。
- **月/季/年三级视图切换**——按月（当月每日）、按季（连续 4 个月）、按年（全年 365 天），`‹` `›` 导航切换。
- 暖色散点（就寝）与冷色散点（起床）在同一时间尺上，每夜以**细垂直线段**连接。
- **分层预计算性能优化**——所有夜晚数据预计算，视图切换仅 filter + 算术，搭配 React `startTransition` 并发渲染。
- 统计卡：时长 / 就寝 / 起床 / 天数。

**年度热力图**
- **双视图切换**：年度视图 / 近 365 天（标题区胶囊切换器，默认近 365 天）。
- GitHub 贡献图范式，**响应式 Grid**（`aspect-ratio: 1`），一屏完整显示无横向滚动。
- 6 分类 pill 切换——点击时全局颜色平滑过渡（`--c-active` CSS 变量驱动）。
- **强度算法：百分位动态阈值**（25%/50%/75%/90%），基于数据分布自动分级。
- 格子 hover 放大 1.6 倍 + 1px 黑色边框；今日格恒久黑色 box-shadow。
- 统计栏（累计 / 日均 / 目标 / 连续 / 最佳）含连续 Pip 可视化。
- 年度视图：`‹` `›` 切换年份，月份标签 1 月~12 月。
- 滚动视图：固定 53 周铺满，月份标签跨年显示，右箭头在最新时禁用。
- 两行标题结构：第一行时间范围（28px），第二行标题主体（14px 斜体），翻页箭头垂直居中。

**数据成熟度：** Cold / Warming / Mature 三级生效。

### ICS 导入

- **解析 RFC 5545 文件**（通过 ical.js）。自动跳过全天事件和重复事件，预览中显示数量。
- **事件名聚合**——导入后按事件名聚合（1369 行 → ~28 种类型），每行显示事件名 + 事件次数。给"吃饭"分一次类自动应用到所有 487 条记录。
- **行内分类按钮**——6 个颜色圆点按钮（编号 1-6），单击即分类，支持键盘 1-6 快速分配。
- **智能预填建议**——基于关键词匹配自动猜分类，虚线边框 + 图标提示。**一键应用 N 条建议**批量接受高置信度猜测。
- **事件覆盖度进度条**——显示"已覆盖 743 / 1369"（按事件数，而非按类型数）。先分类高频事件进度条立刻前跳，正反馈强烈。
- **搜索过滤**——事件名搜索框，30+ 类型时快速缩小范围。
- **同名单例覆盖**——展开按钮支持对同名事件中的个别实例单独覆盖。
- **关键词修改自动重归类**——修改关键词后全库事件自动重新匹配。

### 生活洞察（v3.22）

把每日上下文和作息数据关联起来，回答"是什么影响了我"。

- **分组对比分析**——按每个变量（社交、运动、饮食时间等）将历史数据分为高值组和低值组，对比两组作息指标的均值差异。
- **自然语言洞察卡片**——分析结果以白话呈现，如"较晚的最后一餐：就寝时间推迟了 34 分钟"。
- **相关性 ≠ 因果**——每条洞察附带免责声明，避免用户误判。
- **建议方向**——对显著关联提供可操作的微调建议，如"尝试睡前 1 小时减少屏幕使用"。
- **日视图入口**——任意日期点击「🧠 洞察」按钮，分析最近 60 天的上下文数据。

### 稳态指标（v3.22）

从「连续天数」的冲刺心态转向「长期稳定」的稳态视角。

- **睡眠中位数**——就寝时间和起床时间的中位数，而非均值，抗异常值。
- **标准差**——就寝和起床时间的波动幅度，衡量作息稳定性。
- **漂移速度**——通过线性回归计算作息时间每周变化量（分钟/周），正数=推迟，负数=提前。
- **漂移投影**——"若不干预，4 周后就寝时间将到达 XX:XX"。
- **统计页入口**——Stats 页面新增「稳态」标签，展示全部稳态指标卡片。

### 克制模式（v3.22）

防止工具异化为新的焦虑源。允许用户在某些时段只记录、不被分析、不被对照、不被警告。

- **一键开关**——设置 → 数据 → 克制模式。
- **生效范围**——开启后 AnalyzeButton（✨）隐藏，AI 对话和洞察入口不可见。
- **设计定位**——帮助用户看见生活方式的因果链，而非持续审视用户。

### AI 时间助手

右侧滑入面板，把你的时间数据变成对话。支持多 AI 提供商（DeepSeek / OpenAI / Claude / 自定义兼容）。

- **@ 提及**——输入 `@` 快速引用分类、事件、日期或周。提及自动解析为结构化上下文注入提示词，事件的 description 同时注入上下文——写过的描述 AI 看得见。
- **Tab 标题同步**——浏览器标签页标题动态显示 "CaILens · 2026 第 20 周" 等，方便多 tab 用户定位。
- **会话状态恢复**——重开应用恢复上次浏览的周、滚动位置和视图（周/日/统计），localStorage 持久化。
- **日历联动**——点击日历事件或框选时间段，自动带入对话作为引用上下文。事件卡片和日记条目上出现「Ask AI」按钮。
- **反向锚定**——AI 提到"Python 学习"等关键词时，鼠标 hover 对应文字，日历中匹配的事件块保留不透明度，非匹配事件变暗，一眼定位。
- **Markdown 渲染**——加粗、列表、引用块、代码块、表格正常渲染，不再显示为纯文本。
- **内联数据可视化**——回复中的百分比自动转为迷你进度条，时间数字转为彩色标签，对比数据渲染为微型柱状图（基于 recharts）。
- **上下文自适应按钮**——对话底部按钮不再固定，根据对话状态动态变化：空白时提供开场问题 → 分析后有深入追问 → 聊天后可请求本周分析。
- **历史按周归档**——历史图标点开后，会话按周分组可折叠，含日期范围和会话数，支持搜索过滤，自动提取话题摘要。
- **AI 事件反链**——AI 回复中的事件名称会转为可点击链接 `[事件标题](event:ID)`，点击跳转到日历对应事件。
- **分析沉淀到日记**——Pin 按钮将重要 AI 分析固定到当天的日视图，成为可回顾的资产。日视图中可取消固定。
- **反馈系统**——每条 AI 回复下有 👍 / 👎 / 🔄 按钮，评价数据存储于本地，既是用户控制权，也是未来 prompt 优化的数据来源。

> AI 子系统的提示词架构设计详见 [`docs/ai-design/prompt-architecture.md`](./docs/ai-design/prompt-architecture.md)。

### 数据

- **本地持久存储**——IndexedDB（Dexie v4）。Schema 已到 version 9。迁移自动执行。
- **连续记录统计**——`computeStreak()` 计算连续有记录的周数。
- **数据导出**——CSV、JSON、**加密 .cailens 导出（age 加密 + gzip 压缩）**。
- **数据导入**——CSV、JSON 事件恢复。
- **加密备份**——.cailens 格式：gzip 压缩 + passphrase 加密（age-encryption），完整的导入/导出双向流程。

## 怎么跑起来

需要 **Node 20 以上** 和 **npm**。

```bash
git clone https://github.com/vanemacus486-bit/CaILens.git
cd CaILens
npm install
npm run dev
```

打开 Vite 打印的地址（通常是 `http://localhost:5173`）。

## 架构

```
domain/  ──→  data/  ──→  stores/  ──→  features/ + components/
 (纯逻辑)    (Repository)   (Zustand)        (UI)
```

- **`domain/`** —— 纯类型和业务逻辑。不依赖 React、不依赖 IndexedDB、没有副作用。全单元测试覆盖。
- **`data/`** —— Dexie schema 和 repository。项目里唯一直接碰 IndexedDB 的地方。
- **`stores/`** —— Zustand store，包装数据层。组件永远不直接调 repository。
- **`features/` + `components/`** —— React UI。`features/` 放业务视图；`components/` 放可复用原语。

几个值得一提的实现：

- **拖拽系统**——分钟轴坐标系（0..7×1440），跨天拖拽只是加减法。ghost 浮层基于 rAF 直接操作 DOM，不触发 React 重渲染。
- **渲染性能**——`React.memo`、稳定 callback、Zustand 切片订阅，拖一个事件不会让整周重渲染。
- **快速记录**——`n` 键全局快捷键 + `QuickLogDialog` 表单。默认时间和颜色由 `deriveDefaultTimes` / `deriveDefaultColor` 纯函数推导，全局快捷键 hook 独立可复用。
- **统计引擎**——纯函数构成的周统计、时段聚合、区间合并、Type I/II 分离、连续记录、年度推演、数据成熟度、回顾生成、偏差分析、记录质量指标。
- **AI 集成**——流式 SSE 多提供商、提示词构建器（系统 + 画像 + 技能 + 自定义）、上下文注入（提及 + 日历选区）、对话存储与归档、分析沉淀系统。
