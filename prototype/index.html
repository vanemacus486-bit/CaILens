设计决策记录

  • 侧边栏导航：从顶部水平导航改为左侧垂直图标栏，释放垂直空间，更符合桌面应用习惯。
  • 表面层级：三层表面（base / raised / sunken）替代扁平布局，卡片和面板有清晰边界。
  • 色彩克制：延续 #F1EBE0 暖基色 + #c47a5a 锈色强调。强调色每屏最多两处。
  • 信息层级：页面标题 → 分区标题 → 卡片标题 → 正文，字号逐级递减，对比明确。
  • 桌面窗口感：自定义标题栏 + 状态栏 + 原生滚动条，模拟 Tauri/Windows 窗口体验。
  • 事件色不变：6 种事件分类色（accent/sage/sand/sky/rose/stone）保持语义，不随主题变化。
